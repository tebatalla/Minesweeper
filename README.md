# Ruby Minesweeper
![minesweeper]
Command line Minesweeper. To start, clone the repo, `bundle` and run
```
$ ruby minesweeper.rb
```
### Features
- Save/load game
- Quick reveal (hit R on a square bordering a flag when you know there's no more bombs around that square)
